## 🚀 如何运行项目

### 后端启动
```bash
cd /home/myproject01/backend
# 安装依赖
pip install -r requirements.txt
# 数据库迁移
alembic upgrade head
# 启动服务
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
# 运行测试
pytest tests/ -v
```

### 小程序开发
```bash
# 1. 使用微信开发者工具打开项目
# 项目路径: /home/myproject01/frontend/mini-app/src

# 2. 配置小程序 AppID
# 编辑 project.config.json 设置 appid

# 3. 配置后端 API 地址
# 编辑 api/request.ts 中的 baseURL
```

### 商家后台启动
```bash
cd /home/myproject01/frontend/admin
npm install
npm run dev
```

## 💡 开发提示

1. **API 基础地址**: 默认 `http://localhost:8000/api`，需要在 `api/request.ts` 中配置
2. **微信小程序 AppID**: 需要在 `project.config.json` 中配置真实的 AppID
3. **后端测试**: `pytest tests/ -q` 可以快速运行所有测试（346个测试全部通过）
4. **组件使用**: 新页面可以直接引入已有的通用组件
5. **TabBar 图标**: 目前图标路径是占位，需要准备真实的图标文件（81x81px PNG）
